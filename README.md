# 🍎 Apple-Inspired Glassmorphism Calculator

A premium, minimalist web calculator built with HTML, CSS, and JavaScript. This project explores high-end UI/UX principles, specifically the "Glassmorphism" aesthetic popularized by macOS.

## ✨ Features
- **Glassmorphism UI**: Uses CSS `backdrop-filter` for a modern blurred-glass effect.
- **Precision Logic**: Custom JavaScript engine for accurate arithmetic operations.
- **Keyboard Support**: Fully mapped for physical keyboard usage (Enter, Backspace, Escape).
- **History Tracking**: Integrated log to view recent calculations.
- **Responsive Design**: Optimized for both mobile and desktop browsers.

## 🛠️ Tech Stack
- **HTML5**
- **CSS3** (Custom Variables & Grid)
- **JavaScript** (ES6+)
